# 🚀 Antigravity Furniture UI

A premium **futuristic furniture e-commerce homepage** built using **Next.js, Tailwind CSS, and Framer Motion**, inspired by antigravity UI principles and cinematic design aesthetics.

---

## ✨ Overview

This project explores a **next-generation furniture shopping experience** with:

* Floating UI elements (antigravity feel)
* Glassmorphism & layered depth
* Smooth scroll-based animations
* Premium dark luxury design system
* Interactive product showcase

Designed primarily as a **UI/UX concept project** for portfolio presentation.

---

## 🎯 Features

* 🧊 **Glassmorphism UI** with depth and blur
* 🪶 **Antigravity floating elements**
* 🎬 **Scroll-triggered animations**
* 🎯 **Interactive product cards (expand/collapse behavior)**
* 🧠 **Stacked review section with scroll reveal**
* 🎨 **Dark luxury theme with gold accents**
* ⚡ **Smooth micro-interactions & hover effects**

---

## 🧩 Sections Included

* Hero Section (Dashboard-style UI)
* About Section (Layered image composition)
* Products Section (Interactive cards)
* Editorial Section (Asymmetric layout)
* Testimonials / Reviews (Stacked scroll reveal)
* Footer (Minimal premium layout)

---

## 🛠 Tech Stack

* **Framework:** Next.js (App Router)
* **Styling:** Tailwind CSS
* **Animations:** Framer Motion
* **Icons:** Lucide React

---

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Govindh07/Antigravity_UI.git
cd antigravity-ui
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open in browser:

```
http://localhost:3000
```
