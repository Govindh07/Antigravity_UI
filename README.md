# Antigravity_UI

A premium, high-end, futuristic e-commerce UI toolkit and dashboard built with Next.js, React, and Framer Motion. 
Features a cinematic sticky-scrolling stacked card layout and buttery-smooth animations.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
